📱 App 配置指南

在 WebBridgeKit App 中设置服务器地址为：

http://<服务器IP>:8765

将上方的地址填入 App 的「服务器配置」页面即可。服务器将自动处理推送、口令和清单请求。

Bark 兼容说明：本服务器兼容 Bark 推送协议，可直接将 Bark 客户端的服务器地址指向此服务器。

📋 Curl 示例

健康检查：
curl http://localhost:8765/health
发送推送：
curl -X POST http://localhost:8765/push \
  -H "Content-Type: application/json" \
  -d '{"device_key":"YOUR_KEY","title":"测试","body":"Hello"}'
生成口令：
curl -X POST http://localhost:8765/api/v1/commands \
  -H "Content-Type: application/json" \
  -d '{"type":"urlScheme","data":"myapp://action"}'
Bark 兼容：
curl "http://localhost:8765/YOUR_KEY/标题/内容"
