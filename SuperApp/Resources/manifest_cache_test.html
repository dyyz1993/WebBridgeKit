📋 缓存工作流程

  1. HTML 使用相对路径：<img src="logo.png">
  2. baseURL 设置为 custom://
  3. 相对路径自动补全：logo.png → custom://logo.png
  4. WKURLSchemeHandler 拦截 custom:// 请求
  5. 从 manifest.json 查找真实 URL
  6. 下载资源并缓存到本地
  7. 返回资源给 WebView
