# Demo App 功能与 UI 列表

## 概述

Demo App 是 WebBridgeKit 框架的完整演示应用，展示了框架的所有核心功能和最佳实践。应用采用 Tab Bar 架构，包含 4 个主要模块。

---

## 应用架构

### Tab Bar 结构
- **首页** (MainViewController) - 宫格式应用列表
- **用例** (ManifestTestCasesViewController) - 测试用例管理
- **管理** (ManagementViewController) - 收藏与缓存管理
- **设置** (SettingsViewController) - 应用设置中心

---

## 一、首页模块 (MainViewController)

### 功能概述
首页是应用的核心入口，以宫格形式展示用户访问过的 Web 应用历史记录。

### UI 组件
1. **导航栏**
   - 左侧：二维码扫描按钮
   - 标题：首页（大标题样式）
   - 右侧：
     - 存储空间显示标签
     - 清空全部按钮（红色垃圾桶图标）

2. **宫格视图 (CollectionView)**
   - 2 列布局
   - 每个卡片显示：
     - 应用图标/域名首字母
     - 应用标题
     - 域名信息
     - 缓存大小
     - 访问次数
     - 置顶图标（如已置顶）
     - 收藏图标（如已收藏）
   - 支持下拉刷新

3. **分组显示**
   - 置顶应用（如有）
   - 收藏应用（如有）
   - 最近访问
   - 每个分组有独立的标题栏

4. **空状态视图**
   - 图标：宫格图标
   - 标题："开启您的极速体验"
   - 描述："扫描二维码或输入 URL 即可体验离线加载"

### 核心功能

#### 1. 应用访问
- **点击卡片**：打开对应的 Web 应用
- **长按卡片**：显示操作菜单
  - 打开
  - 置顶/取消置顶
  - 收藏/取消收藏
  - 清除缓存
  - 取消

#### 2. 二维码扫描
- 扫描 Manifest URL：自动解析并加入缓存
- 扫描普通 URL：直接打开网页
- 扫描自定义协议：
  - `wb-app://load?url=xxx` - 加载并缓存
  - `wb-app://open?url=xxx` - 直接打开
  - `wb-app://test-cases` - 跳转到测试用例页面

#### 3. 历史记录管理
- 自动记录访问历史
- 显示访问次数
- 显示缓存大小
- 支持置顶功能
- 支持收藏功能
- 支持清除单个缓存
- 支持清空全部历史

#### 4. 口令识别
- 自动检测剪贴板中的口令
- 弹窗提示是否打开
- 支持自定义口令格式

#### 5. 上次应用记忆
- 可选功能（在设置中开启）
- 启动时自动打开上次访问的应用
- 无动画直接进入

#### 6. 存储空间显示
- 实时显示总缓存大小
- 格式化显示（KB/MB/GB）

## 二、测试用例模块 (ManifestTestCasesViewController)

### 功能概述
用于测试和验证 Manifest 缓存功能，展示所有测试用例并支持批量运行。

### UI 组件
1. **导航栏**
   - 标题：测试用例
   - 右侧：运行全部按钮

2. **表格视图**
   - 每个测试用例卡片显示：
     - 测试名称
     - 测试描述
     - 测试状态（待运行/运行中/成功/失败）
     - 运行按钮
     - 查看日志按钮（测试完成后）

3. **顶部提示**
   - 说明文字："点击运行按钮执行测试，测试结果会记录到日志文件"

4. **空状态视图**
   - 图标：文档图标
   - 标题："暂无测试用例"
   - 描述："请添加测试用例到 test_resources 目录"

### 核心功能

#### 1. 测试用例管理
- 自动扫描 `test_resources` 目录
- 显示所有可用测试用例
- 支持单个运行
- 支持批量运行（依次执行）

#### 2. 测试执行
- 显示运行状态
- 记录测试结果
- 生成日志文件
- 支持查看详细日志

#### 3. 日志查看
- 使用 QuickLook 预览日志文件
- 支持分享日志
- 记录详细的测试过程

## 三、管理模块 (ManagementViewController)

### 功能概述
合并了收藏管理和缓存管理两个功能，通过顶部分段控件切换。

### UI 组件
1. **导航栏**
   - 标题视图：分段控件（收藏 / 缓存）
   - 左右按钮根据当前页面动态变化

### 3.1 收藏管理 (FavoriteViewController)

#### UI 组件
1. **导航栏**
   - 右侧：添加按钮

2. **表格视图**
   - 每个收藏项显示：
     - 应用图标
     - 应用标题
     - URL 地址
     - 置顶图标
     - 缓存模式开关
   - 支持下拉刷新
   - 支持拖拽排序

3. **空状态视图**
   - 图标：星星图标
   - 标题："暂无收藏"
   - 描述："点击右上角添加收藏，或浏览网页时收藏页面"

#### 核心功能
- 查看所有收藏
- 添加新收藏
- 删除收藏（左滑）
- 置顶/取消置顶
- 开启/关闭缓存模式
- 拖拽排序
- 点击打开应用

### 3.2 缓存管理 (CacheManagementViewController)

#### UI 组件
1. **导航栏**
   - 右侧：全部清除按钮

2. **表格视图**
   - 每个缓存应用显示：
     - 应用图标
     - 应用名称
     - AppID
     - 缓存大小
     - 页面数量
     - 最后更新时间
   - 支持下拉刷新
   - 支持左滑操作

3. **底部统计**
   - 总缓存大小
   - 应用数量

4. **空状态视图**
   - 图标：托盘图标
   - 标题："暂无缓存"
   - 描述："加载网页时会自动创建缓存"

#### 核心功能

##### 1. 缓存列表
- 显示所有已缓存的应用
- 实时更新缓存大小
- 显示页面数量
- 显示最后更新时间

##### 2. 缓存操作
- **左滑菜单**：
  - 删除：删除整个应用缓存
  - 详情：查看应用详细信息
- **长按菜单**：
  - 复制 AppID
  - 查看详情
  - 删除缓存

##### 3. 应用详情 (CacheAppDetailViewController)
- 显示应用基本信息
- 显示所有缓存页面列表
- 每个页面显示：
  - 页面 Key
  - 缓存大小
  - 资源数量
- 支持删除单个页面缓存
- 支持查看页面资源详情

##### 4. 资源详情 (CacheResourceViewController)
- 显示页面的所有缓存资源
- 每个资源显示：
  - 资源 URL
  - 资源类型
  - 文件大小
- 支持复制资源 URL

1. **顶部信息卡片**
   - 应用图标

2. **设置列表**

#### 2. 服务器配置 (ServerConfigViewController)

- **API 端点配置**（自定义模式）

- **操作按钮**

##### 核心功能

- 实时验证输入

- 保存配置

2. **临时密钥列表**

     - 密钥名称

3. **Bark 推送配置**

   - 保存配置按钮

4. **空状态视图**
   - 图标：时钟图标
   - 标题："暂无临时密钥"
   - 描述："点击右上角 + 按钮创建临时密钥"

##### 核心功能

###### 永久密钥
- 显示永久密钥（脱敏）
- 复制密钥到剪贴板
- 刷新密钥（需确认）
- 测试推送功能
- 查看使用示例

###### 临时密钥

- 查看所有临时密钥
- 复制临时密钥
- 删除临时密钥（左滑）
- 测试临时密钥推送（左滑）
- 编辑群组绑定（左滑）

###### Bark 推送配置
- 配置 Bark Key
- 配置服务器地址
- 保存配置
- 用于开发调试推送功能

###### 使用示例 (APIKeyExampleViewController)
- 显示密钥使用方法
- 提供代码示例
- 说明 API 调用方式

#### 4. 缓存管理入口
- 显示总缓存大小

#### 5. 上次应用记忆
- 开关控件

## 总结

Demo App 是一个功能完整、设计精美的演示应用，全面展示了 WebBridgeKit 框架的能力：

- **4 个主要模块**：首页、测试用例、管理、设置
- **20+ 个视图控制器**：覆盖所有功能场景
- **30+ 个自定义 UI 组件**：统一的设计语言
- **完整的数据管理**：历史、收藏、缓存、密钥、消息
- **丰富的交互方式**：点击、长按、左滑、拖拽、扫码
- **优秀的用户体验**：流畅的动画、清晰的反馈、友好的引导

这个 Demo 不仅是框架功能的展示，也是最佳实践的参考实现。
