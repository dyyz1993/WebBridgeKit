🚀 Manifest Cache System Demo

Demonstrating relative path loading with custom:// URL scheme

📋 How It Works

This page uses relative paths for all resources:

  • Images: <img src="logo.png">
  • CSS: <link href="styles.css">
  • JS: <script src="app.js">

These relative paths are automatically resolved to custom:// URLs and intercepted by the URL scheme handler!

🎯 Custom URL Scheme

Uses custom:// instead of http:// for complete control

📦 Manifest-Based

Maps relative paths to real network URLs via manifest.json

⚡ On-Demand Loading

Resources fetched only when needed, then cached

🔄 Transparent

No HTML modifications required after initial setup
